# 🎥 YouTube Trending Video Analytics (SQL)

## 📌 Project Overview

This project analyzes YouTube Trending Video data using SQL to uncover insights into content performance, audience engagement, and trending patterns. The analysis focuses on key metrics such as views, likes, comments, categories, and publishing trends to understand what drives video popularity.

## 🎯 Objectives

* Analyze trending video performance across different categories.
* Identify the most viewed and most engaged videos.
* Compare engagement metrics such as likes, comments, and view counts.
* Discover publishing trends and category-wise popularity.
* Generate business insights using SQL queries.

## 🛠️ Tools & Technologies

* **SQL**
* **MySQL / PostgreSQL** (depending on your setup)
* **YouTube Trending Dataset**

## 📊 Key Analysis Performed

* Top-performing videos by views.
* Category-wise trending video analysis.
* Most liked and commented videos.
* Engagement rate calculations.
* Publishing date and trend analysis.
* Channel performance comparison.
* Ranking videos using SQL window functions.
* Aggregate analysis using GROUP BY, HAVING, and subqueries.

## 🧠 SQL Concepts Used

* SELECT Statements
* Filtering (WHERE)
* Aggregations (SUM, AVG, COUNT)
* GROUP BY & HAVING
* CASE Statements
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions (RANK, DENSE_RANK, ROW_NUMBER)
* Joins
* Date Functions

## 📈 Sample Insights

* Entertainment and Music categories consistently generate the highest view counts.
* Videos with higher engagement rates tend to remain on the trending list longer.
* Certain channels dominate trending sections due to consistent audience interaction.
* Publishing time and category influence trending performance.

3. Review the outputs to generate insights and visualizations.
